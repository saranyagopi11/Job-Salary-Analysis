# 💼 Job Salary Prediction Dashboard | Power BI

## 📌 Project Overview

This project is an interactive Power BI dashboard designed to analyze job salary data and provide insights into salary trends across different job roles, experience levels, locations, and employee demographics.

The dashboard enables users to explore salary distributions, compare job categories, and identify factors that influence employee salaries through interactive visualizations.

---

## 🎯 Objectives

- Analyze salary trends across different job roles.
- Compare salaries based on experience level.
- Identify high-paying job categories.
- Visualize employee distribution by various factors.
- Provide business insights using interactive dashboards.

---

## 📊 Dashboard Features

- 📈 Salary Trend Analysis
- 💰 Average Salary by Job Role
- 📊 Salary Distribution
- 🥧 Category-wise Analysis
- 📍 Geographic Salary Insights
- 📋 Detailed Data Table
- 🎯 KPI Cards
- 🔍 Interactive Filters (Slicers)
